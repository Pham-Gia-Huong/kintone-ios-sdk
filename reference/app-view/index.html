    BuiltinType

    Name Type Value Description
    ASSIGNEE String ASSIGNEE The "Assigned to me" View.
    (This list is automatically created if the Process Management settings have been enabled in the app.)

    ViewType

    Name Type Value Description
    LIST String LIST List View
    CALENDAR String CALENDAR Custom View
    CUSTOM String CUSTOM Custom View
